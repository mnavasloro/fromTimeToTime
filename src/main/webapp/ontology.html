From Time To Time

Ontology to convert time-related formats



The From Time to Time Ontology (ft3:) is an ontology that allows the representation of different temporal expressions and event annotation formats in the form of linked data. Additionally, it facilitates the conversion among formats.




This work is part of the PhD thesis by María Navas-Loro at the Ontology Engineering Group (Universidad Politécnica de Madrid). Please visit her website or the GitHub repository for more information.
